# ghost-scan

> Port scanner + service fingerprinter

Think `nmap` but with better output formatting and JSON-native design.

## Planned Features

- TCP/UDP port scanning
- Service version fingerprinting
- JSON output mode (`--json`)
- CIDR range support
- Export to CSV / Markdown report
